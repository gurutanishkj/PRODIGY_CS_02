# PRODIGY_CS_02

This project is a simple Image Encryption and Decryption Tool developed as part of the Prodigy InfoTech Cyber Security Internship (Task 02). The application uses pixel manipulation techniques to secure digital images by modifying their RGB pixel values using a user-defined numeric key. A graphical user interface (GUI) built with Tkinter allows users to easily select an image, enter an encryption key, and perform encryption or decryption operations. During encryption, the key value is added to each RGB component of every pixel, while during decryption the same key is subtracted to restore the original image. The project supports multiple image formats including PNG, JPG, JPEG, and BMP, and also provides a preview feature to display selected and processed images within the application. This project helped in understanding image processing concepts, pixel manipulation techniques, encryption and decryption fundamentals, GUI development using Tkinter, and image handling with the Pillow (PIL) library. The application is developed using Python 3, Tkinter, and Pillow, making it a simple yet effective demonstration of basic image encryption techniques in cybersecurity.
